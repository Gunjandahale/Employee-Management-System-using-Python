# 🧑‍💼 Employee Management System (Python)

A **menu-driven Employee Management System** developed using **Python** that helps manage employee records, attendance, and salary reports. The project uses **text files** for permanent data storage and demonstrates the practical implementation of **Python File Handling**, **Functions**, **Conditional Statements**, **Loops**, and the **OS Module**.

---

## 📌 Features

✅ Add Employee  
✅ View All Employees  
✅ Search Employee by ID  
✅ Update Employee Details  
✅ Delete Employee Record  
✅ Department-wise Employee List  
✅ Mark Attendance  
✅ View Attendance Records  
✅ Generate Salary Report  
✅ File Handling using `.txt` files

---

## 📁 Data Storage

### employees.txt

Stores employee information in the following format:

```
EmployeeID,Name,Age,Gender,Department,Designation,Salary,Phone,Email
```

Example:

```
101,Rahul Sharma,25,Male,IT,Developer,45000,9876543210,rahul@gmail.com
```

---

### attendance.txt

Stores attendance records.

```
Date,EmployeeID,Status
```

Example:

```
15-07-2026,101,Present
```
